Tutorial: an introduction to GAMMs in intonation modeling

This tutorial provides a basic introduction to using Generalized Additive Mixed Models in intonation research, with some basic concepts, exploring how GAMMs are fit, and an example modeling intonational tunes.

See here for a stand-alone introduction with examples and images.

And for the hands-on vesion, see here to download the code as an RMarkdown document.
